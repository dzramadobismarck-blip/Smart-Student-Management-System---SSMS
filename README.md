# Smart Student Management System - Multipage Version

Java Swing desktop application for managing student profiles, attendance, results, fees, and reports across separate pages.

## Login

- Username: `admin`
- PIN: `2026`

## Pages

- Login
- Home
- Students
- Attendance
- Results
- Fees
- Reports

Each working page includes navigation buttons such as `Home`, `Students`, and `Reports`.

## Features

- Add, update, delete, and search students
- Record daily attendance
- Add subject results
- Record fee payments
- Track fee balance
- Calculate average score
- Calculate attendance percentage
- Generate smart status and advice
- Export a complete student report
